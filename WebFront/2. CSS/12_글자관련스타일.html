color: 글자색 지정

        선택자 {
            color : 색상명 
                    | 16진수 숫자 rgb 
                    | rgb(255, 255, 255)
                    | rgba(255, 255, 255, 1) -> a는 투명도(0:투명, 1:불투명 - 소수점으로 조절)
                    | hsl(360, 100, 100) -> 색상, 채도, 명도 
                    | hsla(360, 100, 100, 1) 

            * 색과 관련된 모든 CSS 속성에서 사용 가능
        }
  • 색상명으로 지정
  • 16진수로 지정
  • rgb로 지정
  • rgba로 지정
  • hsl로 지정
  • hsla로 지정
