글꼴(폰트) 관련 스타일

font-size : 글꼴의 크기 지정

        px : 픽셀(해상도)
        pt : 포인트(1pt = 0.72인치)
        % : 상위 요소 크기에 대한 백분율
        em : 1em == 100%

font-weight : 글꼴의 굵기(두께) 지정

        숫자 : 100 ~ 900(100단위) (100쪽으로 갈수록 얇음)
        normal : 기본 두께(400)
        bold : 굵게(700)

        bolder : 상속 받은 두께보다 더 굵게
        lighter : 상속 받은 두께보다 더 얇게
