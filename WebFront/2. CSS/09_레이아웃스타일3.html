배치 관련 스타일(position)

        position은 요소의 위치를 지정하는 속성

        position : relative; 
        - 지정된 요소 내부에 기준을 만드는 속성

        - 내부에 작성되는 요소에 위치 지정시 
          top, bottom, left, right 속성 사용 가능 

        position : absolute;
        - 기본 요소의 배치 순서를 무시하고 지정된 위치에 요소 배치   
    
        position : fixed;
        - 고정된 곳에 요소 배치(e.g. 따라다니는 배너)
