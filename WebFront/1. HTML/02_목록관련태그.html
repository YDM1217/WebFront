02_목록관련태그

ul태그(unorder list) : 순서없는 목록을 작성하는 태그

ol태그(oredered list) : 순서있는 목록을 작성하는 태그

li태그 : 목록에 들어가는 내용을 작성하는 태그

    ul태그
  • 김밥
  • 서브웨이
  • 백반
    ol태그
  1. 김밥
  2. 서브웨이
  3. 백반
   
        ul태그 기본값 : disc
        style = "list-style-type : ?" 로 변경 가능


        ol태그의 기본값 : 1부터 1씩 증가

        * ol태그의 type속성 *
        "1", "a", "A", "i(로마자)", "I(로마자)"

        * 추가 속성
        start="5"  -> 5부터 시작
        reversed  -> 반대로
